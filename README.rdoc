== Overview

A game. With squares. Endless possibility.

== Development

Requires Coffeescript (https://github.com/jashkenas/coffee-script).

To compile all coffeescript files into js/lib:
coffee -b -o js/lib/ -c js/src/

And to compile all coffeescript files in realtime while coding:
coffee -b -w -o js/lib/ -c js/src/
